# Qubic vorstellen: Die Zukunft der Kryptowelt

:::info

Die Qubic-Dokumentation befindet sich derzeit in der öffentlichen Vorschau und kann sich aufgrund von Rückmeldungen von Lesern wie Ihnen erheblich ändern. Treten Sie dem Qubic [Discord](https://discord.gg/2vDMR8m) bei, um Ihr Feedback im #docs-Kanal zu teilen.

:::

Machen Sie sich bereit, in Qubic einzutauchen – eine innovative Krypto-Plattform, die die Spielregeln verändert. Gegründet von Sergey Ivancheglo, dem Mastermind hinter IOTA und NXT, revolutioniert Qubic die Welt der Kryptowährungen mit seinem quorum-basierten Computersystem (QBC).

Im Kern wird Qubic von 676 [Computors](/learn/nodes) betrieben, die für die Ausführung von Smart Contracts und die Sicherung des Netzwerks verantwortlich sind. Die Plattform gewährleistet Zuverlässigkeit, indem ein [Quorum](/learn/quorum) von ⅔+ (oder 451+) Computors einen Konsens erreichen muss, bevor dieser als endgültig betrachtet wird. Im Gegensatz zu traditionellen Krypto-Plattformen, die enorme Energiemengen für das Mining verbrauchen, verfügt Qubic über ein [Useful Proof-of-Work](/learn/upow)-System, das Mining-Kapazitäten für KI-Training nutzt.

Neben Computors umfasst Qubic auch Kandidaten, die zum KI-Training beitragen, ohne dafür eine Vergütung zu erhalten. Ein wöchentliches Ranking, oder "Epoch", bestimmt die 451 leistungsstärksten Computors, die ihren Status behalten, während die verbleibenden Plätze aus dem Pool der Kandidaten und niedriger eingestuften Computors besetzt werden.

Jede Epoch generiert 1 Billion [Qubic Units](/learn/tokenomics) (QUs), die hauptsächlich unter den Computors verteilt werden. Computors haben außerdem die Möglichkeit, Initiativen, von denen sie glauben, dass sie dem Qubic-Ökosystem zugutekommen, zu unterstützen, indem sie einen Prozentsatz ihrer Einnahmen für diese bereitstellen. Ineffiziente Computors und Miner sehen einen Teil ihrer Belohnungen verbrannt, während der Rest an den [Arbitrator](/learn/arbitrator) geht. Bemerkenswert ist, dass die Rolle des Arbitrators auf die Verwaltung von KI-Trainingsaufgaben, die Beilegung von Streitigkeiten und den Schutz von Nutzerinteressen beschränkt ist und keinen Einfluss auf Smart Contracts, Abstimmungen oder die QU-Verteilung hat.

Qubic zeichnet sich durch gebührenfreie Transaktionen aus, eine Seltenheit unter Krypto-Plattformen. Während die Ausführung von Smart Contracts in der Regel eine Provision verursacht, stimmen Computors per Quorum über die Höhe der Provision ab, die dann verbrannt wird, um Inflation und Deflation zu regulieren.

Dank der direkten "Bare-Metal"-Codeausführung von Qubic verfügt es über die schnellsten Turing-vollständigen Smart Contracts, die in C++ geschrieben sind. Im Gegensatz zu virtualisierungsbasierten Plattformen ermöglicht Qubics Ansatz die volle Nutzung der Rechenleistung, was die Ausführung von Smart Contracts in nur wenigen Sekunden ermöglicht.

Im Wesentlichen vereint Qubic die besten Eigenschaften bestehender Plattformen und beseitigt deren Einschränkungen, was es zur idealen Wahl für diejenigen macht, die eine zuverlässige, effiziente und benutzerfreundliche Krypto-Plattform suchen.
